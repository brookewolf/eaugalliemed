## 🎯 Website Overview

A modern, premium, mobile-first medical website built for Eau Gallie Medical Center. The design is comparable to Mayo Clinic, Cleveland Clinic, and Baptist Health while maintaining a warm, approachable, and community-focused feel.

## ✨ Features

### Design
- ✅ Mobile-first responsive design
- ✅ Premium, clean aesthetic with navy blue and teal accents
- ✅ Large, modern typography
- ✅ Rounded cards with soft shadows
- ✅ Smooth scroll animations
- ✅ WCAG AA color contrast compliance
- ✅ Accessibility best practices

### Pages & Sections
- **Home** - Hero section with prominent call-to-action
- **About Us** - Company background and values
- **Services** - Service offerings (Urgent Care, Family Medicine, Weight Loss, Physicals)
- **Insurance & Payments** - Payment options and information
- **FAQ** - Frequently asked questions with accordion
- **Testimonials** - Patient reviews and ratings
- **Contact** - Location, phone, hours, appointment form
- **Map** - Google Maps embed

### Interactive Features
- 🎯 Sticky navigation bar
- 📱 Mobile-responsive menu
- 💬 Contact form with validation
- 📞 Click-to-call buttons
- 🗺️ Google Maps integration
- ❓ Expandable FAQ accordion
- ✨ Scroll animations
- 📊 Animated stat counters

### Technical
- 🔍 SEO optimized
- 📋 Structured data (MedicalClinic schema)
- 🎨 Clean, maintainable HTML/CSS/JavaScript
- ⚡ Fast loading
- 📄 Meta tags and Open Graph
- 🌐 Semantic HTML5
- ♿ WCAG AA compliant

## 🔧 Customization

### Update Contact Information
Edit the contact section in `medical-index.html`:
- Address
- Phone number
- Hours
- Email

### Change Colors
Modify CSS variables in `medical-styles.css`:
```css
:root {
    --primary: #YOUR_COLOR;
    --accent: #YOUR_COLOR;
}
```

### Add Services
Add new service cards in the Services section with:
- Icon (Font Awesome)
- Title
- Description
- Link

### Customize Hours
Update the FAQ accordion answer for "What are your hours?" and the footer.
